# Bistro Lasagne Maestro – statická webstránka

## Súbory
- `index.html` – obsah stránky
- `style.css` – vzhľad a responzivita
- `script.js` – mobilné menu, lasagne karty, prepínanie denného menu a WhatsApp rezervácia
- `assets/` – všetky obrázky

## Rýchla aktualizácia menu
Týždenné menu aktualizujete prepísaním:
- `assets/menu-tyzdenne.png`

Denné menu aktualizujete prepísaním:
- `assets/menu-pondelok.png`
- `assets/menu-utorok.png`
- `assets/menu-streda.png`
- `assets/menu-stvrtok.png`
- `assets/menu-piatok.png`

Názvy súborov musia zostať rovnaké. Potom zmeny odošlite do GitHub repozitára.

## Spustenie
Stačí otvoriť `index.html`, alebo projekt publikovať cez GitHub Pages / Netlify / Vercel.
